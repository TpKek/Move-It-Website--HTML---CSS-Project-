## ğŸš€ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server required - this is a static website

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/Move-It-Website.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd Move-It-Website
   ```

3. **Open the website:**
   - Simply open `index.html` in your web browser
   - Or serve it with a local server for better development experience

### Local Development Server

For a better development experience, you can serve the files locally:

**Using Python (if installed):**
```bash
python -m http.server 8000
```

**Using Node.js (if installed):**
```bash
npx serve .
```

Then open `http://localhost:8000` in your browser.
